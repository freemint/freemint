Tsetitimer()

LONG Tsetitimer(which, interval, value, ointerval, ovalue)

WORD which;
LONG *interval;
LONG *value;
LONG *ointerval;
LONG *ovalue;
Tsetitimer() schedule an interval timer.
Opcode329 (0x0149)
AvailabilityAvailable when a 'MiNT' cookie with a version of at least 1.11 exists.
Parameterswhich is ITIMER_REAL (0) for SIGALRM, ITIMER_VIRTUAL (1) for SIGVTALRM, or ITIMER_PROF (2) for SIGPROF.

The rest of the parameters are pointers to millisecond values. interval is the value to which the timer will be reset. value is the current timer value. ointerval and ovalue are the previous values.

Binding 
pea	ovalue
pea	ointerval
pea	value
pea	inteval
move.w	which,-(sp)
move.w	#$0149,-(sp)
trap	#1
lea	$12(sp),sp
Return ValueTsetitimer() returns E_OK or a negative GEMDOS error code otherwise.
CommentsSetting new interval values needs root privileges.
