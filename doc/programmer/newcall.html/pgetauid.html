Pgetauid()

LONG Pgetauid(VOID)

Pgetauid() returns the audit user id for the calling process.
Opcode325 (0x0145)
AvailabilityAvailable when a 'MiNT' cookie with a version of at least 1.11 exists.
Binding 
move.w	#$0145,-(sp)
trap	#1
addq.l	#2,sp
Return ValuePgetauid() returns the auid on success or a negative GEMDOS error code otherwise.
CommentsAudit id never changes once set to != 0 and can therefore be used to determine what id was originally attributed to the user at the begin of the current session.
See Also Psetauid(), Pgetuid(), Pgetreuid()
